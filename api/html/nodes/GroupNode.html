GroupNode¶

class NodeGraphQt.GroupNode(qgraphics_views=None)¶

The NodeGraphQt.GroupNode class extends from the :class:NodeGraphQt.BaseNode class with the ability to nest other nodes inside of it.

Inherited from: NodeGraphQt.BaseNode

collapse()¶

Collapse the group node session it’s expanded child sub graphs.

See also

NodeGraph.collapse_group_node(), SubGraph.collapse_group_node().

expand()¶

Expand the group node session.

See also

NodeGraph.expand_group_node(), SubGraph.expand_group_node().
