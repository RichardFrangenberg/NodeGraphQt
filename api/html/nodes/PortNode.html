PortNode¶

Port node are the nodes in a expanded NodeGraphQt.SubGraph that represent the ports from the parent NodeGraphQt.GroupNode object.

PortInputNode¶

class NodeGraphQt.nodes.port_node.PortInputNode(qgraphics_views=None, parent_port=None)¶

The PortInputNode class is the node object that represents a port from a NodeGraphQt.GroupNode when expanded in a NodeGraphQt.SubGraph.
